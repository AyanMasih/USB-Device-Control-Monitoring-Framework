# USB Device Control & Monitoring Framework

## Overview
This project focuses on monitoring USB device activity on a system.
It demonstrates how endpoint security solutions detect removable
devices and log USB activity for security auditing.

This project is created for learning and internship purposes only.

---

## Features
- USB device detection
- Continuous monitoring
- Safe and non-intrusive
- Beginner-friendly Python implementation
